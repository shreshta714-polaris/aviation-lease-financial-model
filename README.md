# Aviation Lease Cash Flow & Valuation Model

## Overview

This project is a comprehensive aviation leasing financial model built in Excel to evaluate aircraft lease investments, debt financing structures, and investor returns.

The model simulates lease cash flows, debt amortization, residual value assumptions, and key investment metrics used in aircraft leasing transactions.

## Business Problem

Aircraft leasing companies invest millions of dollars in aircraft acquisitions and need to determine whether a lease transaction generates attractive returns while maintaining healthy debt coverage ratios.

This model helps answer:

- Is the aircraft lease economically viable?
- What return can equity investors expect?
- Can the lease cash flows support debt obligations?
- How sensitive are returns under different operating scenarios?

## Model Features

### Aircraft Lease Analysis

The model evaluates multiple aircraft leasing scenarios, including:

- Boeing 737-800
- Airbus A320neo

Key assumptions include:

- Aircraft purchase price
- Lease rate factor
- Monthly lease payments
- Lease term
- Residual value assumptions
- Lessee credit quality

### Debt Financing Module

The financing model incorporates:

- Loan-to-Value (LTV) assumptions
- Debt amortization schedules
- Interest expense calculations
- Principal repayment tracking

### Cash Flow Modeling

The model projects:

- Monthly lease revenue
- Operating cash flows
- Debt service obligations
- Cash Flow Available for Debt Service (CFADS)
- Equity cash flows

### Scenario Analysis

Users can evaluate multiple scenarios:

- Base Case
- Upside Case
- Downside Case

Scenario selection dynamically updates financial projections and investment returns.

## Key Metrics Calculated

- Equity Internal Rate of Return (IRR)
- Multiple on Invested Capital (MOIC)
- Debt Service Coverage Ratio (DSCR)
- Minimum DSCR
- Total CFADS
- Residual Value Proceeds
- Loan Balance Schedule

## Dashboard

An executive dashboard summarizes:

- Equity returns
- Debt coverage metrics
- Loan statistics
- Residual value estimates
- Scenario performance

allowing stakeholders to quickly evaluate investment attractiveness.
